# purin_mamma

`purin_mamma`는 아기 이유식 식단표를 달력으로 계획하고, 식재료별 알레르기 테스트와 실제 섭취 기록을 함께 관리하기 위한 프로젝트입니다.

## 주요 방향

- 월간/주간 이유식 식단표 생성
- 신규 식재료 도입 및 알레르기 관찰 일정 관리
- 식재료 변경 시 이후 식단, 테스트 일정, 장보기 목록 자동 갱신
- 보호자가 자동 변경 사항을 검토하고 되돌릴 수 있는 변경 이력 제공


## 웹으로 시작하는 이유

초기 제품은 GitHub Pages에 올릴 수 있는 정적 웹페이지로 시작합니다. 링크 하나로 가족과 공유하기 쉽고, 휴대폰 홈 화면에 추가하면 앱처럼 접근할 수 있으며, 이후 필요해지면 PWA나 네이티브 앱으로 확장할 수 있습니다.

## 로컬 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://127.0.0.1:8000/`에 접속해 확인합니다.

## GitHub Pages 배포

1. GitHub 저장소의 **Settings > Pages**로 이동합니다.
2. **Build and deployment**에서 `Deploy from a branch`를 선택합니다.
3. 배포 브랜치와 `/ (root)` 폴더를 선택합니다.
4. 저장 후 표시되는 `https://<username>.github.io/<repository>/` 주소로 접속합니다.

## 의료 주의

이 프로젝트는 보호자의 식단 계획과 기록을 돕는 도구이며 의료 진단이나 치료를 대체하지 않습니다. 알레르기 고위험군이거나 의심 반응이 있는 경우 반드시 소아청소년과 또는 알레르기 전문의와 상담해야 합니다.
